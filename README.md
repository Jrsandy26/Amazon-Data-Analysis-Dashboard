## Project Overview
This project presents a comprehensive sales analysis dashboard for Amazon products, featuring interactive visualizations and key performance indicators (KPIs) to provide actionable business insights. The dashboard analyzes sales data to track revenue performance, product categories, and sales trends.

## Technologies Used
- **Data Source**: Excel files (raw sales data)
- **Data Processing**: Excel for initial data preparation, Power BI Power Query for data transformation and modeling
- **Visualization**: Microsoft Power BI
- **Data Modeling**: Power BI DAX formulas and calculated measures

## Data Structure
The analysis is based on Amazon sales data including:
- Product information (category, name, price)
- Sales transactions (date, quantity, revenue)
- Customer data (reviews, ratings)
- Time-series data for trend analysis

## Installation & Setup
1. Clone the repository
```bash
git clone https://github.com/Jrsandy26/Amazon-Data-Analysis-Dashboard.git
cd amazon-sales-dashboard
```

2. Requirements
- Microsoft Power BI Desktop (free download from Microsoft)
- Excel files with Amazon sales data

3. Load the data
- Open Power BI Desktop
- Import the Excel files from the `/data/raw/` folder
- Use Power Query to clean and transform the data as needed

4. Open the dashboard
- Open the `.pbix` file in Power BI Desktop
- Refresh data connections if needed
- Publish to Power BI Service (optional) for sharing


## Usage
1. Open the dashboard file
2. Use the filter options to explore different product categories
3. Navigate between different views to analyze sales performance
4. Export insights and reports as needed
